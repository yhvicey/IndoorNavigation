# File define
A path file is a txt file with a specified format.

# File example
```
1,10,10
1,20,10
1,30,10
1,40,10
2,10,10
2,20,10
2,30,10
2,40,10
```

# Structure
```
<FloorIndex>,<X>,<Y>
[,<FloorIndex>,<X>,<Y>]
...
```

# Element defines

|Element|Description|
|---|---|
|FloorIndex|Floor index of this node.|
|X|X position of the node in double. Start from 0.|
|Y|Y position of the node in double. Start from 0.|
